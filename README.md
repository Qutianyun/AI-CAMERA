# AI_CAMERA_PWA_V2_AI_MEMORY

## 功能

- 1:4:1 手机全屏布局，不需要上下滑动。
- 顶部四个模式按钮：英文、日本、韩文、随机。默认随机。
- 中部为摄像头区域，含打开摄像头、关闭摄像头、拍照识别三个图标按钮。
- 支持放大/缩小：点击右侧 + / -，也支持双指缩放。
- 底部只显示三个结果：判断、语言、风险。
- 拍照后才识别，不做实时连续识别。
- AI辅助学习：当选择英文/日本/韩文并识别OK时，系统会把样本特征保存到本机浏览器 localStorage。后续识别相似内容时自动降低风险。

## 部署

把本文件夹内全部文件上传到 GitHub Pages 仓库根目录，替换旧版本文件即可。

## 注意

- 摄像头需要 HTTPS 环境。
- OCR 使用 Tesseract.js CDN，首次识别需要联网加载。
- 本地AI记忆保存在当前手机浏览器，不同手机不共享。
