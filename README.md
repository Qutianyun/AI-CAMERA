# AI_CAMERA_PWA_V3_FAST_AI

## 本版重点

1. 顶部模式选择区和底部结果区高度减少一半，摄像头区域显著加大。
2. 底部增加“时间”，显示从点击拍照识别到结果输出的耗时。
3. 识别完成后自动回到实时摄像头画面，不再停留在上一次拍照照片。
4. 识别算法从 OCR 优先改为“已知产品模板快速匹配”，目标是接近 1 秒内出结果。
5. 已内置本次上传的英文 / 日本 / 韩文三张样张特征。
6. 保留 AI 辅助学习：非随机模式下识别 OK 且低风险时，把当前样本特征保存到本机浏览器，后续相似样本风险会降低。

## 注意

- 该版本是针对当前三类产品样张的快速分类算法。
- 如果后续产品版面变化很大，需要继续补充样张并重新生成模板。
- 本地 AI 记忆保存在当前手机浏览器 localStorage，不同手机不共享。
